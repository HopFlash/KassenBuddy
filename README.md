# KassenBuddy

KassenBuddy ist ein offline-faehiges Kassensystem (SvelteKit PWA) fuer den schnellen Verkauf an Theke und Event.

## Features

- Schnelle POS-Oberflaeche mit Produktgrid und Warenkorb
- Prozent-Rabatte als Rabattposition
- Konfigurierbare Schnellrabatt-Buttons im Adminbereich
- Kontingent-Zaehler fuer verkaufte Getraenke (Rabatte ausgenommen)
- Adminbereich fuer Produkte, Statistik und Einstellungen
- Backup/Restore fuer Produkte und Einstellungen

## Entwicklung

```sh
npm install
npm run dev
```

## Qualitaet pruefen

```sh
npm run check
```

## Build

```sh
npm run build
npm run preview
```
